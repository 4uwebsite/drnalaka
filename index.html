Security
No HTTPS (Hypertext Transfer Protocol Secure)

HTTPS uses encryption to secure the data exchanged between the browser and the website. HTTPS is important to a website for several reasons:

Data Security: HTTPS encrypts data transmitted between the user's browser and the website's server, preventing unauthorized access to sensitive information.

User Privacy: HTTPS helps protect the privacy of users by ensuring that their interactions with the website are secure and cannot be intercepted by third parties.

Trust and Credibility: Websites that use HTTPS are considered more trustworthy by users because they demonstrate a commitment to security and protecting user data.

SEO Benefits: Search engines like Google prioritize websites that use HTTPS in their search rankings, providing an SEO boost to websites that implement it.

Compliance: In many jurisdictions, regulations such as the General Data Protection Regulation (GDPR) mandate the use of HTTPS for websites that handle sensitive user data.

Overall, HTTPS is essential for maintaining the security, privacy, and integrity of data exchanged between users and websites on the internet.

UI/UX (User Interface / User Experience)
Overuse of Animations

Animations should be strategically used. Currently almost every element on the page loads with an animation. Ideally, you only want to animate elements that should grab the user’s attention; i.e. CTA (Call to Action) elements, important content, situations where user input/interaction is required. Overuse of animations makes the page look complicated to the user and fails to distinguish between what should/shouldn’t grab the user’s attention.

Typography

Menus/Headings font-family (stack): Poppins, sans-serif

Subheadings/Body font-family (stack): Quicksand, sans-serif

Inconsistent hierarchical typography in the website gives poor UX. Good typographical hierarchy helps the users intuitively distinguish the structure of the content.

Sizing & Spacing

When it comes to sizing and spacing elements on a webpage, it's essential to prioritize content hierarchy and readability. This is particularly crucial for medical websites, where users seek valuable information rather than visual appeal. If the layout and organization of information are confusing or overwhelming, users are more likely to leave the website, resulting in increased bounce rates. Therefore, it's important to ensure that the structure of the page is clear and easy for users to understand, enhancing their overall experience and keeping them engaged with the content.

SEO (Search Engine Optimization)
Empty Alt Tags

Alt tags (alternative text), are important for SEO for several reasons:

Accessibility: Alt tags provide a textual description of images on a webpage, making them accessible to visually impaired users who use screen readers to browse the web. This is especially important for a website related to the medical field.

Image SEO: Search engines cannot interpret images in the same way humans do. Alt tags provide search engines with contextual information about the content of the images, helping them understand what the images are about. This improves the chances of the images appearing in relevant search results.

Keyword Optimization: Alt tags offer an opportunity to include relevant keywords related to the content of the webpage.

Duplicate Content

In some pages the same content is repeated. This is considered bad SEO and search engines often categorise this as keyword loading, leading to poor rankings.

Accordions used for FAQs

This is not a good technique when the invisible accordion body content is SEO rich.

YouTube Embedding and Metadata

When embedding YouTube videos, you would ideally want to include metadata related to the video. This creates a stronger link (from SEO perspective) between YouTube and the website, which intern enhances SEO.

Poor HTML Page Naming

Modern search engines (like Google) utilises LLMs (Large Language Models) popularly known as AI. These systems are context aware. For example, the “About” page on the website is named “aboutus.html” while the page essentially talks about an individual (Dr. Gunawansa). This leads to search engine misclassification issues.

Poorly Implemented Photo Gallery

Photo galleries offer a great opportunity for SEO, if the photos are properly implemented and metadata included. In this case, it is poorly implemented.

Performance
Bootstrap

Bootstrap is a free and open-source front-end framework for developing responsive and mobile-first layouts. While it is useful for large dynamic websites, for a static website of this scale, it is overkill. The Bootstrap framework is large and this website only uses a very small portion of it. This means a considerable percentage of the website’s assets are unused. This negatively affects page load speeds, thereby reducing performance.
